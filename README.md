# macOS Frontier | Web

The professional Choice for macOS deployment. Engineered for experts, refined by the community.

## Overview

macOS Frontier is a high-performance, reliable tool designed for macOS deployment. This repository contains the landing page and web infrastructure for the Frontier ecosystem.

## Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Runtime**: [Bun](https://bun.sh/)

## Getting Started

First, install the dependencies:

```bash
bun install
```

Then, run the development server:

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
